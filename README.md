# Popover Widget

[![Build Status](https://github.com/e1ectron1k/popover-widget/workflows/Build%20and%20Deploy/badge.svg)](https://github.com/e1ectron1k/popover-widget/actions)
[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-blue)](https://e1ectron1k.github.io/popover-widget/)

Реализация виджета Popover на чистом JavaScript без использования jQuery.

## Особенности

- Чистый JavaScript (без зависимостей)
- Адаптивное позиционирование
- Поддержка нескольких popover'ов на странице
- Автоматическое закрытие при клике вне элемента
- Полностью стилизованный внешний вид

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm start

# Сборка проекта
npm run build

# Запуск тестов
npm test
